定义结构

结构体关键字为struct和结构体名组成。
struct语句定了一个包含多个成员的新的数据类型，struct语句格式如下：

tag是结构体标签（即 新数据类型的名称）。
member-list是结构体内部变量。
variable-list是结构变量，可以指定一个或多个结构变量。 ^942df6
