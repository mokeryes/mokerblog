&，与

含义：

取得被运算的两个值同为 1 的位。

运算方式：
两个位都为1 时，结果才为 1 。

|，或

含义：
获取两个值都为 0 的位。
或者
获取两个值至少存在一个 1 的位。

运算方式：
两个位都为 0 时，结果才为 0 。

示例：

^，异或

含义：
获取两个值的异同情况。

运算方式：
相同得 0 ，相异得 1 。

右移

运算方式：
各二进制位全部右移若干位

  • 对于无符号数，高位补零。
  • 对于有符号数，各编译器处理方法不一样，有的补符号位（即忽略第一位符号位，其余右移），有的补零（全都补零）。
